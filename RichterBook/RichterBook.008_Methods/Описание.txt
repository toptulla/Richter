 - Конструкторы (ссылочные типы).
Конструкторы - это специальные методы, позволяющие корректно инициализировать экземпляр класса (задать
исходное состояние объекта).
Конструктор экземпляра объекта класса вызывается после того, как будет выделенная память под экземпляр, а
также инициализированны служебные поля (указатель на объект типа, индекс блока синхронизации).

При конструировании объекта ссылочного типа, выделяемая для него память всегда обнуляется до вызова конструктора
экземпляра типа => любые поля не инициализированные конструктором, гарантированно, содержат либо null, либо 0.

Конструкторы экземпляров не наследуются!!! (У класса есть только те конструкторы, которые определены в классе).

Если определить класс без явно заданных конструкторов, то многие компиляторы создадут конструктор по умолчанию,
который просто вызывает конструктор базового класса. Для abstarct классов этот конструктор помечен модификатором
protected, для всех остальных public.

В типе можно объявить несколько кнонструкторов, при этом их сигнатуры и уровни доступа должны различаться.
